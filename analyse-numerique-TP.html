الاستكمال الحدودي (Polynomial Interpolation)

الاستكمال الحدودي هو أسلوب رياضي يُستخدم لتقدير القيم بين نقاط معطاة. يتم بناء متعددة حدود تمر عبر جميع النقاط المعطاة \((x_i, f_i)\) حيث \(i = 0, 1, \ldots, n\).

1. طريقة لاغرانج (Lagrange Interpolation)

تعتمد طريقة لاغرانج على بناء متعددة حدود من الدرجة \(n\) تمر بجميع النقاط المعطاة.

\[ P(x) = \sum_{k=0}^{n} f_k \cdot L_k(x) \]

حيث \(L_k(x)\) هي متعددات حدود لاغرانج الأساسية وتعطى بالعلاقة:

\[ L_k(x) = \prod_{\substack{j=0 \\ j \ne k}}^{n} \frac{x - x_j}{x_k - x_j} \]
دالة لاغرانج في MATLAB 
function p = lagrange(x, f, v)
% دالة لاغرانج للاستكمال الحدودي
% x: نقاط x المعطاة
% f: قيم الدالة عند نقاط x
% v: النقاط المراد استكمال قيم الدالة عندها (اختياري)

n = length(x);
syms t;
p = 0;

for i = 1:n
    L = 1;
    for j = 1:n
        if i ~= j
            L = L * (t - x(j)) / (x(i) - x(j));
        end
    end
    p = p + L * f(i);
end

p = expand(p);

if nargin > 2
    t = v;
    p = eval(p);
end

end

مثال توضيحي:

لنفترض الدالة \(f(x) = x^2 - 3x + 2\) والنقاط \(x_0 = 0, x_1 = 1, x_2 = 2\).

تنفيذ المثال في MATLAB 
% تعريف النقاط المعطاة
x = [0, 1, 2];
f = [2, 0, 0];

% استدعاء دالة لاغرانج
p = lagrange(x, f);
disp('كثير حدود لاغرانج:');
disp(p);

% تقييم كثير الحدود عند نقطة محددة
value = lagrange(x, f, 0.5);
disp(['القيمة عند x=0.5: ', num2str(value)]);
كثير حدود لاغرانج: t^2 - 3*t + 2 القيمة عند x=0.5: 0.7500

2. طريقة نيوتن (Newton Interpolation)

تستخدم طريقة نيوتن الفروق المقسومة لبناء متعددة الحدود الاستكمالية.

\[ \begin{aligned} P(x) &= f[x_0] \\ &\quad + f[x_0, x_1](x - x_0) \\ &\quad + f[x_0, x_1, x_2](x - x_0)(x - x_1) \\ &\quad + \cdots \\ &\quad + f[x_0, x_1, \ldots, x_n](x - x_0)(x - x_1) \cdots (x - x_{n-1}) \end{aligned} \]
دالة نيوتن للفروق المقسومة في MATLAB 
function p = newton_interpolation(x, f, v)
% دالة نيوتن للاستكمال باستخدام الفروق المقسومة
% x: نقاط x المعطاة
% f: قيم الدالة عند نقاط x
% v: النقاط المراد استكمال قيم الدالة عندها (اختياري)

n = length(x);
% إنشاء جدول الفروق المقسومة
F = zeros(n, n);
F(:,1) = f';

for j = 2:n
    for i = 1:n-j+1
        F(i,j) = (F(i+1,j-1) - F(i,j-1)) / (x(i+j-1) - x(i));
    end
end

syms t;
p = F(1,1);
product = 1;

for i = 1:n-1
    product = product * (t - x(i));
    p = p + F(1,i+1) * product;
end

p = expand(p);

if nargin > 2
    t = v;
    p = eval(p);
end

end

طرق إيجاد جذور المعادلات غير الخطية

المعادلات غير الخطية هي معادلات تحتوي على حدود ذات قوى و/أو حاصل ضرب متغيرات. في معظم الحالات، لا يمكن حل هذه المعادلات باستخدام الطرق الجبرية البسيطة وتتطلب تقنيات عددية أو حلول تقريبية.

1. طريقة التنصيف (Bisection Method)

تعتمد طريقة التنصيف على مبرهنة بولزانو: إذا كانت الدالة \(f\) متصلة على الفترة \([a,b]\) و \(f(a)\) و \(f(b)\) بإشارتين مختلفتين، فإن هناك جذرًا واحدًا على الأقل في \((a,b)\).

دالة طريقة التنصيف في MATLAB 
function root = bisection(f, a, b, tol, max_iter)
% طريقة التنصيف لإيجاد جذر المعادلة f(x) = 0
% f: الدالة المراد إيجاد جذرها
% a, b: حدود الفترة [a, b]
% tol: مقدار التحمل (الدقة)
% max_iter: الحد الأقصى لعدد التكرارات

if f(a) * f(b) >= 0
    error('يجب أن تكون f(a) و f(b) بإشارتين مختلفتين');
end

iter = 0;
while (b - a) / 2 > tol && iter < max_iter
    c = (a + b) / 2;
    fc = f(c);
    
    if fc == 0
        break;
    end
    
    if f(a) * fc < 0
        b = c;
    else
        a = c;
    end
    
    iter = iter + 1;
end

root = (a + b) / 2;
fprintf('تم إيجاد الجذر بعد %d تكرارات\n', iter);

end

مثال توضيحي:

لحل المعادلة \(f(x) = x e^x - \pi = 0\) في الفترة \([0, 2]\) بدقة \(0.0001\):

تنفيذ المثال في MATLAB 
% تعريف الدالة
f = @(x) x .* exp(x) - pi;

% تطبيق طريقة التنصيف
root = bisection(f, 0, 2, 1e-4, 100);
disp(['الجذر التقريبي: ', num2str(root)]);

% التحقق من النتيجة
f_value = f(root);
disp(['قيمة الدالة عند الجذر: ', num2str(f_value)]);
تم إيجاد الجذر بعد 14 تكرارات الجذر التقريبي: 1.0737 قيمة الدالة عند الجذر: -1.2257e-05

2. طريقة نيوتن-رافسون (Newton-Raphson Method)

هي طريقة تكرارية سريعة التقارب (تقارب تربيعي) لإيجاد جذور المعادلات.

\[ x_{i+1} = x_i - \frac{f(x_i)}{f'(x_i)} \]
دالة نيوتن-رافسون في MATLAB 
function [root, iter] = newton_raphson(f, df, x0, tol, max_iter)
% طريقة نيوتن-رافسون لإيجاد جذر المعادلة f(x) = 0
% f: الدالة المراد إيجاد جذرها
% df: مشتقة الدالة f
% x0: التخمين الأولي
% tol: مقدار التحمل (الدقة)
% max_iter: الحد الأقصى لعدد التكرارات

iter = 0;
x = x0;

while iter < max_iter
    fx = f(x);
    dfx = df(x);
    
    if abs(dfx) < eps
        error('المشتقة قريبة من الصفر، الطريقة قد لا تتقارب');
    end
    
    x_new = x - fx / dfx;
    
    if abs(x_new - x) < tol
        break;
    end
    
    x = x_new;
    iter = iter + 1;
end

root = x_new;

if iter == max_iter
    warning('تم الوصول إلى الحد الأقصى لعدد التكرارات');
end

end

مثال توضيحي:

لحل المعادلة \(f(x) = e^x - \pi x = 0\) باستخدام طريقة نيوتن-رافسون:

تنفيذ المثال في MATLAB 
% تعريف الدالة ومشتقتها
f = @(x) exp(x) - pi * x;
df = @(x) exp(x) - pi;

% تطبيق طريقة نيوتن-رافسون
x0 = 0.5; % تخمين أولي
[root, iter] = newton_raphson(f, df, x0, 1e-6, 100);

disp(['الجذر التقريبي: ', num2str(root)]);
disp(['عدد التكرارات: ', num2str(iter)]);

% التحقق من النتيجة

disp(['قيمة الدالة عند الجذر: ', num2str(f_value)]);
الجذر التقريبي: 0.55383 عدد التكرارات: 4 قيمة الدالة عند الجذر: 2.2204e-16

التكامل العددي (Numerical Integration)

التكامل العددي هو تقنية تُستخدم لتقريب التكامل المحدد لدالة ما على فترة معينة. يُستخدم التكامل العددي عندما يكون من الصعب أو المستحيل الحصول على حل تحليلي للتكامل.

1. قاعدة شبه المنحرف (Trapezoidal Rule)

تقوم هذه القاعدة على تقريب الدالة بخط مستقيم بين كل نقطتين متجاورتين.

\[ \int_a^b f(x) dx \approx \frac{h}{2} [f(a) + 2f(a+h) + 2f(a+2h) + \cdots + 2f(b-h) + f(b)] \]

حيث \(h = \frac{b-a}{n}\)، و \(n\) هو عدد الفترات.

دالة قاعدة شبه المنحرف في MATLAB 
function I = trapezoidal_rule(f, a, b, n)
% قاعدة شبه المنحرف للتكامل العددي
% f: الدالة المراد تكاملها
% a, b: حدود التكامل
% n: عدد الفترات

h = (b - a) / n;
x = a:h:b;
y = f(x);

I = (h/2) * (y(1) + 2*sum(y(2:end-1)) + y(end));

% حساب الخطأ النظري (للدوال التي يمكن حساب مشتقتها الثانية)
if nargout > 1
    % هنا يمكن إضافة حساب الخطأ إذا كانت المشتقة الثانية معروفة
end

end

مثال توضيحي:

لحساب \(\int_0^2 \sin(x) dx\) باستخدام قاعدة شبه المنحرف مع \(n=10\):

تنفيذ المثال في MATLAB 
% تعريف الدالة
f = @(x) sin(x);

% تطبيق قاعدة شبه المنحرف
a = 0;
b = 2;
n = 10;
I_trap = trapezoidal_rule(f, a, b, n);

% القيمة الحقيقية للمقارنة
I_exact = -cos(b) + cos(a);

disp(['نتيجة قاعدة شبه المنحرف: ', num2str(I_trap)]);
disp(['القيمة الحقيقية: ', num2str(I_exact)]);
disp(['الخطأ: ', num2str(abs(I_trap - I_exact))]);
نتيجة قاعدة شبه المنحرف: 1.4114 القيمة الحقيقية: 1.4161 الخطأ: 0.0047396

2. قاعدة سيمبسون (Simpson's Rule)

تقوم هذه القاعدة على تقريب الدالة بقطع مكافئ بين كل ثلاث نقاط متتالية.

\[ \int_a^b f(x) dx \approx \frac{h}{3} [f(a) + 4f(a+h) + 2f(a+2h) + 4f(a+3h) + \cdots + 4f(b-h) + f(b)] \]

حيث \(h = \frac{b-a}{n}\)، و \(n\) يجب أن يكون عددًا زوجيًا.

دالة قاعدة سيمبسون في MATLAB 
function I = simpsons_rule(f, a, b, n)
% قاعدة سيمبسون للتكامل العددي
% f: الدالة المراد تكاملها
% a, b: حدود التكامل
% n: عدد الفترات (يجب أن يكون زوجيًا)

if mod(n, 2) ~= 0
    error('عدد الفترات n يجب أن يكون زوجيًا');
end

h = (b - a) / n;
x = a:h:b;
y = f(x);

% تطبيق قاعدة سيمبسون
I = (h/3) * (y(1) + y(end) + 4*sum(y(2:2:end-1)) + 2*sum(y(3:2:end-2)));

end

مثال توضيحي:

لحساب \(\int_0^2 \sin(x) dx\) باستخدام قاعدة سيمبسون مع \(n=10\):

تنفيذ المثال في MATLAB 
% تعريف الدالة
f = @(x) sin(x);

% تطبيق قاعدة سيمبسون
a = 0;
b = 2;
n = 10;
I_simp = simpsons_rule(f, a, b, n);

% القيمة الحقيقية للمقارنة

disp(['نتيجة قاعدة سيمبسون: ', num2str(I_simp)]);
disp(['القيمة الحقيقية: ', num2str(I_exact)]);

نتيجة قاعدة سيمبسون: 1.4161 القيمة الحقيقية: 1.4161 الخطأ: 4.2397e-06

مقارنة بين طرق التكامل

قاعدة شبه المنحرف:

  • بسيطة وسهلة التطبيق
  • دقتها منخفضة مقارنة بطرق أخرى
  • مناسبة للدوال الخطية تقريبًا

قاعدة سيمبسون:

  • أكثر دقة من قاعدة شبه المنحرف
  • تتطلب عددًا زوجيًا من الفترات
  • مناسبة للدوال السلسة
